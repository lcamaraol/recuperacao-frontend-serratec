# 📝 Reavaliação Front-End

Este é um projeto feito para apresentar um questionário de avaliação de conhecimentos em Front-End. 
O projeto contém 20 questões de múltipla escolha estruturadas em páginas individuais.

## 🛠️ Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica das páginas e questões.
*   **CSS3:** Estilização com design focado em cards para facilitar a leitura e distinção das alternativas.

## 🚀 Como Executar o Projeto

Este projeto é completamente estático e não requer a instalação de pacotes, dependências ou servidores locais.

1. Baixe ou clone os arquivos do projeto.
2. Va até a pasta raiz do projeto.
3. Selecione o arquivo principal **`index.html`**.
4. Pressione a tecla **F5** para executar a aplicação.
5. Utilize o menu principal para acessar cada uma das questões.

## 📁 Estrutura de Arquivos

*   `index.html`: Arquivo principal contendo o menu de navegação.
*   `style.css`: Arquivo de estilos global aplicado em todo o projeto.
*   `questoes/`: Diretórios e arquivos `.html` contendo as perguntas individuais.
